Software Guide

软件说明

Two Arduino sketches — use certain_angle during assembly calibration, 16_module for full operation.

两个 Arduino 程序——装配校准阶段使用 certain_angle，整机运行使用 16_module。

code/16_module

Controls all 16 servos via PCA9685 with a sinusoidal gait. Each servo's servoSign[] sets its extend/retract phase within the 4×4 array.

通过 PCA9685 以正弦步态控制全部 16 路舵机，每路舵机的 servoSign[] 决定其在 4×4 阵列中的伸缩相位。

code/certain_angle

Sets one servo (pin A3) to a fixed angle via 14-bit PWM at 333 Hz. Use exclusively during assembly calibration — not for multi-servo control.

通过 14 位 PWM（333 Hz）将引脚 A3 上的舵机设置到固定角度。仅用于装配校准阶段，不用于多舵机控制。

Assembly Guide
装配指南

Module Calibration & Assembly

模块校准与装配

Follow this workflow precisely. Identical kinematics across all modules depend on it.

严格遵循以下流程，所有模块的运动学一致性取决于此。

1
Direct Microcontroller Calibration
直接通过微控制器校准
No PCA9685 required at this stage
此阶段无需使用 PCA9685
  • Do NOT use the PCA9685 board. Connect servo signal lines directly to the Arduino Nano ESP32 GPIO pins.
  • 禁止使用 PCA9685 驱动板。 将舵机信号线直接连接至 Arduino Nano ESP32 的 GPIO 引脚。
  • Power source: USB only. Do not connect any external power supply.
  • 供电方式： 仅 USB 供电，暂勿接入任何外部电源。
  • Software: Use certain_angle only. Do not run 16_module yet.
  • 软件控制： 只运行 certain_angle，暂勿运行 16_module。
  • Why? Minimal hardware eliminates I2C errors and PCA9685 configuration issues during mechanical alignment.
  • 原因： 最小化硬件依赖，消除机械对准阶段的 I2C 通信错误及扩展板配置问题。
2
Step-by-Step Calibration & Assembly
逐步校准与装配
Repeat all three steps for every module
每个模块均需重复执行以下三步
1

Servo Initial Zeroing

舵机初始归零

Zero the servo spline before installing it into the module casing to prevent mechanical binding post-assembly.

在装入模块外壳之前将舵机花键归零，防止装配后出现机械卡死。

  1. Connect the servo directly to the Arduino Nano ESP32.
  2. Run certain_angle and set the target angle to −90 degrees.
  3. Wait until the servo fully settles and locks before proceeding.
  1. 将舵机直接连接至 Arduino Nano ESP32。
  2. 运行 certain_angle，将目标角度设为 −90 度。
  3. 等待舵机完全到达并锁定该位置后，再进行下一步。
2

Reference Deconstruction & Observation

参考模块拆解与观察

Use an existing correctly-built module as a visual reference for servo horn alignment.

以一个已验证正常的模块作为摆臂对准的视觉基准。

  1. Carefully disassemble the reference module.
  2. Document the exact relative orientation of servo horn vs. body at −90°.
  3. Note spline tooth engagement to prevent gear-tooth offset in new modules.
  1. 仔细拆解参考模块。
  2. 记录舵机摆臂与本体在 −90° 时的精确相对朝向。
  3. 注意花键齿啮合位置，防止新模块出现齿位偏移。
3

Calibrated Module Assembly

校准后的模块装配

With the servo zeroed and reference alignment known, assemble the physical module.

完成舵机归零和参考对准后，进行实物组装。

  1. Mount the servo horn, duplicating the reference angle observed in Step 2.
  2. Verify internal component consistency — all linkage positions must match across modules.
  3. Tighten all screws to eliminate backlash, rotational slop, and structural shifting.
  1. 安装舵机摆臂，复现步骤 2 中观察到的参考角度。
  2. 核查内部零件一致性——各模块内所有连杆位置必须完全相同。
  3. 拧紧所有螺钉，消除机械间隙、旋转松旷和结构偏移。
⚠️ CRITICAL SAFETY WARNING — POWER SEQUENCE 重要安全警告 — 供电顺序

STOP. Do not connect the battery or proceed to full-system testing until all individual modules have been calibrated, assembled, and verified using USB power only.

停止。在所有单模块均已通过 USB 供电完成校准、装配和验证之前，禁止连接电池或进行整机测试。

Verify that all logic, commands, and servo movements function correctly before connecting the battery. An undetected short circuit under battery power may cause permanent damage to the microcontroller and surrounding electronics.

连接电池前，必须确认所有系统逻辑、控制指令及舵机动作均正常。若存在未发现的短路，电池供电时将导致微控制器及周边电子元件永久损坏。

Only after all modules pass USB-powered testing should the battery be connected.只有当所有的模块在仅仅通过 USB 给 Arduino Nano ESP32 供电调试好之后，才能去连接电池进行下一步。
